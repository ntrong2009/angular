• Tương thích với các trình duyệt cũ

    Một số tính năng JavaScript hiện đại và các API của trình duyệt không được hỗ trợ đầy đủ trên các phiên bản trình duyệt cũ hơn. Polyfills cung cấp các triển khai thay thế để đảm bảo ứng dụng Angular của bạn có thể chạy trên nhiều trình duyệt.

  • Quản lý các polyfills một cách tập trung

    File polyfills.ts tập trung tất cả các polyfills cần thiết ở một nơi, giúp dễ dàng quản lý và bảo trì.

It looks like your application or one of its dependencies is using i18n. Angular 9 introduced a global $localize() function that needs to be loaded.

lỗi xảy ra khi bạn sử dụng các tính năng i18n (quốc tế hóa) trong Angular 9 trở lên nhưng chưa thêm gói @angular/localize vào ứng dụng của mình.

Sau khi cài đặt gói @angular/localize, bạn cần import và chạy hàm $localize trong file polyfills.ts. Điều này đảm bảo rằng hàm này được định nghĩa toàn cục và sẵn sàng sử dụng trong ứng dụng của bạn. 

            import '@angular/localize/init';
